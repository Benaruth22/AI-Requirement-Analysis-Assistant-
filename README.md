# AI Requirement Analysis Assistant

An AI-powered application that automates Software Requirement Specification (SRS) analysis using Large Language Models (LLMs) and Natural Language Processing (NLP).

## Features
- Analyze SRS documents
- Extract functional & non-functional requirements
- Detect ambiguities, inconsistencies, duplicates
- Generate Agile user stories
- Generate acceptance criteria
- Produce structured outputs

git clone https://github.com/Benaruth22/AI-Requirement-Analysis-Assistant.git
